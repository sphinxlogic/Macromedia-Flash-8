Interactive animation: Keys
 
Description
There are many ways you can create interactive motion with Macromedia Flash. This sample demonstrates how you can create a symbol that moves in response to keyboard input.
 
Key features and ActionScript elements
Key object, variables, object properties
 
Instructions
Click anywhere in the movie below. Move the beetle up, down, left, right, or diagonally with the arrow keys on your keyboard. The arrow keys onscreen are only a graphical representation and have no effect when clicked.
 

 
The source file for this movie (movement-keys.fla) can be found in the Flash MX/Samples folder.
