Interactive animation: Follow pointer or go to clicked spot
 
Description
There are many ways you can create interactive motion with Macromedia Flash. This sample demonstrates how you can use ActionScript to make a symbol move either toward the pointer or toward a selected spot.
 
Key features and ActionScript elements
Math object, Boolean variables, conditions(if...else)
 
Instructions
Click anywhere in the recessed area of the movie below to make the beetle travel to that spot. Click the top toggle button to make the beetle follow the pointer. Click the bottom toggle button to change the way the beetle rotates.
 
 
The source file for this movie (Movement-pointer_or_click.fla) can be found in the Flash MX/Samples folder.
