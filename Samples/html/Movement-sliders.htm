Interactive animation: Sliders
 
Description
There are many ways you can create interactive motion with Macromedia Flash. This sample demonstrates how you can create a symbol that moves in response to the position of sliders. The rotation of the symbol is calculated based on the combined values of the two sliders.
 
Key features and ActionScript elements
Draggable movie clips, Math object, variables, object properties
 
Instructions
Manipulate the sliders to make the beetle move.
 

 
The source file for this movie (Movement-sliders.fla) can be found in the Flash MX/Samples folder
