Games: Maze
 
Description
This sample demonstrates how you can use Macromedia Flash to create complex games.
 
Key features and ActionScript elements
Collision detection (hitTest), Key object, tweened motion along a path
 
Instructions
Click Start to begin. Use the arrow keys to navigate to the end of the maze. The computer controlled opponent will travel through the maze in the opposite direction.
 

 
The source file for this movie (Maze.fla) can be found in the Flash MX/Samples folder.
