Interactive animation: Buttons
 
Description
There are many ways you can create interactive motion with Macromedia Flash. This sample demonstrates how you can create buttons that control the motion of a symbol.
 
Key features and ActionScript elements
Buttons, button events, variables, object properties
 
Instructions
Click the buttons to move the beetle up, down, left, or right. Click the toggle button to change the way the beetle rotates.
 

 
The source file for this movie (movement-buttons.fla) can be found in the Flash MX/Samples folder.
