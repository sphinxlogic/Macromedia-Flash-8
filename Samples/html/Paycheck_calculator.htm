Feature highlight: Dynamic text
 
Description
In this sample, the 2001 California State and Federal tax charts have been converted to ActionScript. It demonstrates how you can use ActionScript and dynamic text fields to create forms with dynamic feedback.
 
Key features and ActionScript elements
Input text fields, variables, conditions (if...)
 
Instructions
Enter values in the text fields and notice the breakdown of information in the lower right corner and the net result in the lower left corner. The charts for married people have not been converted to ActionScript, so the marital status toggle is disabled in this sample. This sample is for educational purposes only.
 

 
The source file for this movie (Paycheck_calculator.fla) can be found in the Flash MX/Samples folder.
