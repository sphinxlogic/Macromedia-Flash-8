Feature highlight: Load images
 
Description
This sample demonstrates how you can load images from a drive or server into a movie at runtime. There are no images in the library of the source file for this sample.
 
Key features and ActionScript elements
loadMovie, buttons, Key object, Text Field object, increment(++), decrement(--)
 
Instructions
Load a new image by clicking the Previous and Next buttons or enter a value between 1 and 5 in the Photo # text field and press Enter.
 

 
The source file for this movie (Load_images.fla) can be found in the Flash MX/Samples folder.
