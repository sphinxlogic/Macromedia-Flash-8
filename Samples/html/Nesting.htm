Basics: Movie clip nesting
 
Description
This sample demonstrates how movie clips (children) nested inside other movie clips (parents) inherit the position of their parents. The ActionScript used also demonstrates how you can create valid absolute and relative target paths.
 
Key features and ActionScript elements
Draggable movie clips, buttons
 
Instructions
Drag any of the small square buttons to move the squares and rectangles around as labelled.
 

 
The source file for this movie (Nesting.fla) can be found in the Flash MX/Samples folder.
