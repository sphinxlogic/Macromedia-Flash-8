Feature highlight: Import video and components with Live Preview
 
Description
This sample demonstrates how you can import video into Macromedia Flash. It also contains a dynamic playback controller that will control the timeline of any movie clip or root timeline in which it is placed. The playback controller has been made as a component and the width of the scrub bar can be changed by changing parameters in the Property inspector.
 
Key features and ActionScript elements
Imported video, Timeline properties (_currentframe, _totalframes)
 
Instructions
Click the buttons on the playback controller to play, stop, step through, or go to the beginning or end of the video. If the Loop option is toggled on, the movie will start over when it reaches the last frame, otherwise it will stop.
 

 
The source file for this movie (Import_video.fla) can be found in the Flash MX/Samples folder.
