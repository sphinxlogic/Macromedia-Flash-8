Feature highlight: Scriptable masks
 
Description
This sample demonstrates how you can use two new masking techniques. The first method lets you use ActionScript to designate one movie clip to mask another movie clip. The second new technique lets you use a multi-framed movie clip on a mask layer.
 
Key features and ActionScript elements
setMask
 
Instructions
Click the buttons to switch between different types of scripted masks.
 

 
The source file for this movie (ScriptableMask.fla) can be found in the Flash MX/Samples folder.
