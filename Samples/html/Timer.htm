Feature highlight: getTimer
 
Description
This sample demonstrates how you can use getTimer and dynamic text fields to create a stopwatch-style timer.
 
Key features and ActionScript elements
getTimer, dynamic text fields, Math object, conditions (if...)
 
Instructions
Click the buttons to start, pause, or reset the timer.
 

 
The source file for this movie (Timer.fla) can be found in the Flash MX/Samples folder.
