Interactive animation: Thrust
 
Description
There are many ways you can create interactive motion with Macromedia Flash. This sample demonstrates how you can create a thrusting effect using ActionScript to simulate rudimentary physics.
 
Key features and ActionScript elements
Math object, Key object, if...else
 
Instructions
Click anywhere on the movie below. Press the Left and Right Arrow keys to rotate the rocket-beetle. Press the Up Arrow key to thrust the rocket-beetle in the direction it is facing.
 

 
The source file for this movie (movement-thrust.fla) can be found in the Flash MX/Samples folder.
